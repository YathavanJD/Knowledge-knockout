## 📖 Overview

KnowledgeKnockout is an interactive Android quiz application that challenges players with multiple-choice questions across a variety of topics. Designed with a smooth, intuitive UI, the app makes learning competitive and fun — one question at a time.

---

## ✨ Features

- 🎯 **Multiple-choice gameplay** — Four options per question with instant feedback
- 📊 **Score tracking** — Live score updates throughout the quiz
- 🏆 **Result display** — Final score summary at the end of each round
- 🔄 **Smooth navigation** — Seamless transitions between questions
- 🎨 **User-friendly UI** — Clean, responsive interface built with XML layouts
- 📚 **Educational content** — Questions spanning diverse knowledge areas

## 🚀 Getting Started

### Prerequisites

- [Android Studio](https://developer.android.com/studio) (latest stable version)
- Android SDK (API 21+)
- An Android emulator or physical device

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/KnowledgeKnockout.git
   ```

2. **Open in Android Studio**
   - Launch Android Studio
   - Select **File → Open** and navigate to the cloned folder

3. **Build the project**
   - Let Gradle sync automatically
   - Click **Build → Make Project**

4. **Run the app**
   - Select an emulator or connect a physical Android device
   - Click the **Run ▶** button

---

## 🎮 How to Play

1. Launch the app to reach the home screen
2. Tap **Start Quiz** to begin
3. Read each question and select your answer from the four options
4. Your score updates after every correct answer
5. View your final score and performance summary at the end
